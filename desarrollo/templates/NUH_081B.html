Tipos Garantía

Mediante este formulario se permite la actualización o consulta de las diferentes clases o tipos de garantía ofrecidas o exigidas a los clientes y que pueden avalar las obligaciones por ellos contraídas.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario

Descripción de campos

Garantía

Campo numérico de 3 dígitos, obligatorio, asociado a cada clase o tipo de garantía válida para la entidad.

Descripción

En este campo alfanumérico obligatorio, se registra el nombre o breve descriptivo del tipo o clase de garantía codificada.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es la descripción
