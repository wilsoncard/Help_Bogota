Validaciones 3DSecure

Mediante este formulario la entidad realiza la combinatoria de los tres conjuntos de parámetros previos y, a partir de ese punto le indica al sistema la acción que debe tener lugar, esto es Rechazar o Autorizar la transacción.

Filtro: El formulario cuenta con conjunto de campos o criterios para realizar la búsqueda, mínimo debe contener un dato:

SLI

Campo en el que se puede digitar o seleccionar de la lista de valores provista el código de la SLI3DSecure sobre el cual se desea realizar la consulta.

Versión 3DS

Campo en el que se puede digitar o seleccionar de la lista de valores provista la versión de SLI3DSecure sobre el cual se desea realizar la consulta.

DTI

Campo en el que se puede digitar o seleccionar de la lista de valores provista la variable de riesgo o incertidumbre sobre el cual se desea realizar la consulta.

Acción

Campo que posee lista de valores tipo combo con las opciones Rechazar / Autorizar, una de las cuales se muestra por defecto, sobre el cual se desea realizar la consulta.

El formulario contiene los enlaces: Actualizar, Eliminar, y Adicionar.

Adicionar: Al invocarse la opción Adicionar se despliega un nuevo formulario con los siguientes campos.

SLI

Campo alfanumérico de 3 máximo tres caracteres, obligatorio, en el que se puede digitar o seleccionar de la lista de valores provista, el código de la SLI3DSecure para el cual se desea determinar la validación a realizar y la acción correspondiente.

Versión 3DS

Campo alfanumérico de 3 máximo tres caracteres, obligatorio, en el que se puede digitar o seleccionar de la lista de valores provista, la versión de SLI3DSecure para el cual se desea determinar la validación a realizar y la acción correspondiente.

DTI

Campo alfanumérico de 3 máximo tres caracteres, obligatorio, en el que se puede digitar o seleccionar de la lista de valores provista, la variable de riesgo o incertidumbre para la cual se desea determinar la validación a realizar y la acción correspondiente.

Acción

Campo que posee lista de valores tipo combo con las opciones Rechazar / Autorizar, una de las cuales se debe seleccionar, de tal suerte, que al validarse que se presentan las tres variables antes definidas, el sistema realiza la acción aquí determinada.
