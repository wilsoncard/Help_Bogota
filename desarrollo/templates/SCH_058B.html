Novedad número de acción

En este formulario invocado desde la opción ingreso de novedades, (issuer) la entidad puede modificar el número de acción asociado a la tarjeta cuando esta característica ha sido definido previamente para el producto. Aplica solo para aquellas tarjetas a las cuales se les ha parametrizado a nivel de definición de productos que si Genera Número de Acción, información que es validada por el sistema una vez se ingresa el número de tarjeta y se selecciona esta novedad.

Descripción de campos

Número de acción

Campo obligatorio en el que se registra el número de acción asociado a la tarjeta o crédito.

Fecha de radicación

En este campo obligatorio se registra en formato YYYY-MM-DD la fecha en la cual se solicitó el cambio de número de acción y que puede ser una fecha anterior a la actual del sistema, facilitando el ingreso de novedades de fechas anteriores.
