Tipos de compromisos

Partiendo del concepto que la gestión de cobranza es un proceso de variable complejidad, se provee a través de esta forma o pantalla, la manera de parametrizar las diferentes clases o tipos de compromisos y acuerdos de pago a celebrar con los deudores, de manera que permitan efectuarles un adecuado seguimiento.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo alfanumérico de 3 posiciones, obligatorio, en el que se registra la referencia o identificación de cada uno de los tipos de acuerdo o compromiso.

Descripción

En este campo alfanumérico de 50 posiciones, obligatorio, se ingresa el nombre o breve descripción del acuerdo o compromiso de pago asociado a cada código.
