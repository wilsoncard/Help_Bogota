Tipos de cobro

La principal forma de diferenciar las distintas clases de cobro a realizar a los deudores consiste en definir tantas categorías como requiera la entidad y/o la casa de cobranza. Esta forma o pantalla permite tanto la consulta como la parametrización de los tipos de cobro.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo alfanumérico de 3 posiciones, obligatorio, asignado a cada uno de los distintos tipos de cobro que la entidad defina.

Descripción

En este campo alfanumérico de 50 posiciones, obligatorio, se registra el nombre o descriptivo asociado al código de cada tipo de cobro.

Genera honorarios

Lista de valores con Sí y No en la que selecciona una opción según el caso.
