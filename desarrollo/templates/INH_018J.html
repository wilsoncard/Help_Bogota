Comisión establecimientos

Por medio de esta opción la entidad define las comisiones para los establecimientos, permitiendo parametrizar para cada uno las marcas habilitadas y las terminales.

El formulario contiene las opciones Actualizar y Detalle. Adicionalmente, cuenta con un filtro de búsqueda y los enlaces de Marcas y Terminales.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código establecimiento

Campo numérico de máximo 11 dígitos, por el cual se puede buscar el código asignado a cada una de los establecimientos.

Nombre establecimiento

Campo alfanumérico de 30 caracteres, que permite buscar el nombre del establecimiento.

Nit

Campo numérico de máximo 12 dígitos, por el cual se puede buscar un establecimiento específico a través del Nit.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificablesson: Código de afiliación, Bloqueo sanción, Región, Sub división, Ubicación, y Código de zona.

Detalle: El sistema muestra el siguiente formulario:

Marcas

A través de esta opción el sistema permite parametrizar para cada uno de los establimientos las marcas y las respectivas comisiones que serán.

El formulario contiene las opciones Actualizar, Eliminar, Adicionar y Detalle. Adicionalmente, cuenta con un filtro de búsqueda y un enlace en la parte superior izquierda para ir al menú anterior.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario:

Comisiones

Este formulario permite la parametrización para cada una de las marcas de un tipo de comisión específico discriminado para cada producto y entidad

El formulario contiene la opción Estadísticas y un enlace en la parte superior izquierda para ir al menú anterior.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

Comisión

Lista de valores que permite seleccionar uno de los Tipos de Comisión ya parametrizados.

Producto

Campo obligatorio con lista de valores desde la cual se invoca el producto a relacionar.

Entidad

A través de este campo de lista de valores obligatoria, es posible seleccionar la entidad a la que pertenece

Tipo de comisión

Lista de valores desde la cual es posible seleccionar las siguientes opciones de acuerdo al tipo de comisión parametrizado: Porcentaje, Valor fijo, Ambos, y nulo.

Porcentaje comisión

Este campo sólo se habilita una vez haya sido seleccionada la opción Porcentaje en el Tipo de comisión o Ambos.

Valor comisión

Este campo sólo se habilita cuando en el Tipo de comisión se ha seleccionado la opción Valor fijo o Ambos.
