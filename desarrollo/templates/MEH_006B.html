Esquemas de puntaje

Mediante esta función se definen los parámetros generales aplicables al programa de incentivos al consumo diseñado por la entidad, y específicamente, la administración de los puntos resultantes de dichos consumos; la entidad podrá definir tantos esquemas de puntajes como desee.

Filtro: El formulario cuenta con un filtro inicial de búsqueda:

Código

Campo numérico de máximo 3 dígitos, en el cual se puede ingresar el número asignado o que identifica el esquema de puntos que se desea consultar.
Descripción

En este campo alfanumérico de hasta 30 posiciones, en el cual se puede ingresar el nombre del esquema de puntos, o parte del mismo, que se desea consultar.

Moneda patrón puntaje

Criterio de búsqueda que aplica si, dentro de las compensaciones de los esquemas de puntaje se tiene el abono a la tarjeta, en el cuál se dispone de una lista de valores de la que debe puede seleccionarse la Moneda de referencia para realizar la consulta.

El formulario, además de poder invocar las opciones: Actualizar, Eliminar, y Adicionar, ofrece enlaces a las siguientes opciones: Parámetros, Parámetros adicionales, Parámetros adicionales II ,Orígenes restringidos, Productos y Relaciones.

Adicionar: Si el usuario invoca este enlace se despliega un formulario con los siguientes campos:

Código

Campo numérico de 3 dígitos, obligatorio, en el que se registra el código que identifica el programa o esquema de puntaje.
Descripción

En este campo alfanumérico de máximo 30 posiciones, se digita el nombre o descripción del esquema asociado al código anterior.
Moneda patrón puntaje

Campo con lista de valores de la que debe seleccionarse la Moneda de referencia para evaluar las metas fijadas, facilitando la administración de los puntos.
