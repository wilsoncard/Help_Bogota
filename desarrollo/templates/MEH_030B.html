Definición de otros plásticos

A través de este formulario, la entidad puede parametrizar los diferentes plásticos que adicional a la tarjeta de crédito, se deben realzar y entregar al cliente de acuerdo con lo que se haya definido a nivel del producto

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: si el usuario invoca la opción Adicionar se despliega un nuevo formulario con los siguientes campos.

Descripción de campos

Código

Campo numérico de 11 dígitos, obligatorio, en el cual se registra el código que identifica cada uno de las diferentes plásticos adicionales definidos por la entidad.

Nombre otro plástico

Este campo alfanumérico de 30 posiciones, obligatorio, permite registrar el nombre del otro plástico asociado con el código anterior.

Modelo

Campo obligatorio con lista de valores de la cual se selecciona el modelo que se debe utilizar al momento de efectuar el realce para el plástico que se está parametrizando.

Proveedor

Campo que contiene el nombre del proveedor quien presta los bienes y servicios, quien suministra los plasticos de tarjetas de credito.

Actualizar: si el usuario selecciona un registro e invoca la opción Actualizar, se despliega un formulario en el cual el único campo NO modificable es el Código.
