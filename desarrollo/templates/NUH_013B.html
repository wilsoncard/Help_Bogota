Marcas de vehículos

Mediante esta función se habilita la consulta y mantenimiento de la tabla en la cual se almacenan las diferentes marcas de vehículos existentes en el mercado, con base en la codificación establecida por Fasecolda. La clasificación se hace en tres niveles, siendo la marca al primero de ellos y que corresponde al nombre genérico de fábrica de los vehículos. El segundo nivel corresponde a la Clase de vehículo y el tercer nivel al tipo.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar. Cuenta también con el hipervínculo Clases de vehículo.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

Descripción de campos

Código

Campo alfanumérico de tres dígitos, obligatorio, que contiene el código asignado a cada marca de vehículo.

Descripción

En este campo alfanumérico de 20 posiciones, obligatorio, se registra el nombre de la marca del vehículo asociada a cada código.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es Descripción.

Clases de vehículos: Si el usuario invoca la opción Clases de vehículos, se despliega un nuevo formulario con el mismo nombre y que corresponde al segundo nivel de la clasificación. Permite indicar la característica de los vehículos (automóvil, camioneta, pick up, campero, etc).

El formulario contiene las opciones Actualizar, Eliminar y Adicionar. Cuenta también con el hipervínculo Tipos de vehículos.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

Descripción de campos

Código

Campo alfanumérico de tres dígitos, obligatorio, que contiene el código asignado a cada clase de vehículo.

Descripción

En este campo alfanumérico de 20 posiciones, obligatorio, se registra la clase del vehículo asociada a cada código.

Actualizar: Si el usuario selecciona un registro e invoca la opción Actualizar se despliega un formulario en el cual el único campo modificable es Descripción.

Tipos de vehículos: Si el usuario invoca la opción Tipos de vehículos, se despliega un nuevo formulario con el mismo nombre y que corresponde al tercer nivel de la clasificación. Permite indicar el nombre de las diferentes líneas de vehículos dentro de cada clase y marca.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

Descripción de campos

Código

Campo alfanumérico de tres dígitos, obligatorio, que contiene el código asignado a cada tipo de vehículo.

Descripción

En este campo alfanumérico de 20 posiciones, obligatorio, se registra el tipo del vehículo asociado a cada código.

Actualizar: Si el usuario selecciona un registro e invoca la opción Actualizar se despliega un formulario en el cual el único campo modificable es Descripción.
