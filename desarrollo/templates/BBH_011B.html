Cargue clientes desde CRM

Mediante esta funcionalidad, utilizando la información básica de identificación de un cliente como son Tipo y Número identificación, activar el botón Cargar, si el cliente no existe en la base de datos del aplicativo, permite traer, desde el Core, la data necesaria para crearlo, si por el contrario ya existe, le notifica el usuario.

Descripción De Campos

Tipo identificación

Campo que posee lista de valores tipo combo de la que debe seleccionarse aquel que corresponda al tipo de identificación del cliente que existiendo en el Core no así en la base de datos del aplicativo.

Número identificación

Campo obligatorio, numérico de máximo 16 dígitos, en el que debe registrarse el número de identificación del cliente que existiendo en el Core no así en la base de datos del aplicativo.
