Franquicias

Parámetros acciones para el exception file: Mediante esta función se habilita el mantenimiento y consulta de la tabla que contiene la parametrización requerida por el sistema, para la generación del archivo exception file a enviarse a las franquicias, con las novedades (robos, bloqueos, etc) de las tarjetas y de esa manera mantener actualizado, a nivel internacional, el archivo de Excepción de cada entidad financiera, permitiendo a cada emisor dar una predeterminada respuesta de autorización.

El formulario contiene los enlaces: Actualizar, Eliminar, y Adicionar. Adicionalmente, cuenta con un botón en la parte superior para regresar al formulario principal de Franquicias.

Adicionar: Al activar ese enlace se despliega el siguiente formulario:

Código

Campo alfanumérico, obligatorio, en el que se registra el código mediante el cual cada franquicia identifica las causales o acciones por las que se genera la novedad a ser reportada.

Descripción

Campo alfanumérico de 30 dígitos, obligatorio, en el que se registra el descriptivo asociado a cada código de acción.

Actualizar: Al activar ese enlace se despliega un formulario en el cual el único campo modificable es la Descripción:
