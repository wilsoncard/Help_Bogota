Codigos terminal

Mediante esta función se hace posible especificar, por tipo de terminal, los tipos de dispositivos físicos que podrán conectarse al sistema central de la entidad.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo alfanumérico de 2 posiciones, obligatorio, en el cual se identifica el tipo de terminal.

Descripción

En este campo obligatorio, se registra el nombre de la terminal asociado al código definido anteriormente.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es: Descripción.
