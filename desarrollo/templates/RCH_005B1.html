Actividades

Requisitos por actividad: Mediante esta función se asignan los requisitos correspondientes a cada actividad.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar, y un botón para devolverse a la opción principal Actividades.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Requisito

Campo numérico de 4 dígitos, obligatorio, que contiene el código del requisito que se asignará a la actividad.

Requerido

En este campo obligatorio tipo combo, el usuario puede seleccionar entre SI o NO, según si el requisito que está parametrizando debe ser entregado obligatoriamente cuando el reclamo se encuentre en esta actividad para permitir pasar a la siguiente.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es Requerido.
