Bines internacionales

Esta función habilita la consulta y mantenimiento de la tabla que almacena toda la información necesaria en la validación de transacciones realizadas con tarjetas emitidas en el exterior (Bines internacionales). El mantenimiento se realiza a través del cargue de la cinta enviada periódicamente por el operador (franquicia) y eventualmente de forma manual para resolver movimiento no procesado por falta de actualización de la misma tabla.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Franquicia

Campo obligatorio, con lista de valores de la cual se selecciona la franquicia para la que se define el rango o bin internacional en particular, desplegando el nombre de la misma.

Bin

Campo numérico obligatorio que almacena, en un máximo de 8 caracteres, el Bank Identification Number BIN correspondiente para cada rango de tarjetas.

Rango inferior

En este campo alfanumérico de 16 caracteres, obligatorio, se registra el límite inferior o número de tarjeta a partir del cual se define el rango acorde a cada Bin.

Rango superior

Campo alfanumérico de 16 caracteres, obligatorio, en el que se indica el límite superior o número de tarjeta hasta el cual se define el rango acorde a cada Bin.

Longitud bin

Campo numérico de una posición, obligatorio, que ilustra la longitud del bin dentro del número de la tarjeta.

Longitud tarjeta

En este campo numérico de dos dígitos, obligatorio, se almacena la cantidad máxima de dígitos que componen la cuenta del tarjetahabiente, es decir, el número de la tarjeta.

Tipo

Campo alfanumérico de un dígito, opcional, el cual contiene un indicador asignado por la franquicia a cada bin para identificar el tipo de producto.

Dígito de chequeo

Campo que contiene lista de valores de la que se puede seleccionar el algoritmo mediante el cual se realiza la verificación del dígito de chequeo.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son: Franquicia, Rango superior, Longitud bin, Longitud tarjeta, Tipo, Dígito de chequeo.
