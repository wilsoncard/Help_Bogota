Parámetros de solicitudes

Mediante esta opción se parametrizan las diferentes etapas por las cuales pueden avanzar las solicitudes de productos débito y los documentos que pueden ser requeridos por la entidad en cada una de ellas.

Los datos aquí ingresados serán utilizados posteriormente en la opción de parámetros de solicitudes por producto.

Al ingresar a la opción se despliega un formulario que contiene las etapas actuales parametrizadas mostrando para cada una de ellas el código de la misma, el nombre y el prototipo.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar

Adicionar: si el usuario invoca la opción Adicionar, se despliega un nuevo formulario con los siguientes campos.

Descripción de campos

Código

Campo alfanumérico de hasta 4 caracteres, obligatorio, en el que se registra el código asignado a cada una de las etapas a parametrizar.

Nombre

En este campo alfanumérico de hasta 60 caracteres, obligatorio, se registra el nombre o descripción asociado con cada etapa definida.

Prototipo

Campo tipo combo, obligatorio, que permite seleccionar entre Grabación, Aprobación, Realce y Sobreflex de clave el prototipo asociado a la etapa y que es como internamente la reconoce el sistema.

Actualizar: si el usuario selecciona un registro e invoca la opción Actualizar, se despliega un formulario en el cual el campo modificable es Nombre

Documentos : si el usuario invoca el hipervínculo Documentos, se despliega un nuevo formulario en el que se muestran todos los documentos que la entidad ha definido son exigibles al cliente en cualquiera de las etapas de la solicitud.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar

Adicionar: si el usuario invoca la opción Adicionar, se despliega un nuevo formulario con los siguientes campos.

Descripción de campos

Código documento

Campo numérico de hasta 3 dígitos, obligatorio, en el que se registra el código asignado a cada uno de los diferentes documentos que es posible solicitarle al cliente de acuerdo con las políticas de la entidad.

Nombre documento

En este campo alfanumérico de hasta 60 caracteres, obligatorio, se registra el nombre o descripción asociado con cada uno de los documentos que pueden ser solicitados al cliente.

Actualizar: si el usuario selecciona un registro e invoca la opción Actualizar, se despliega un formulario en el cual el único campo modificable es Nombre documento.
