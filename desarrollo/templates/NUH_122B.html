Tipos de garantía

Desde este formulario se habilita la consulta y mantenimiento de una tabla en la cual se almacenan los códigos de las garantías definidas por las entidades financieras; la forma permite asociarle a cada uno la descripción correspondiente que permitirá una fácil identificación.

El formulario muestra las opciones Actualizar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos :

Descripción de campos

Garantia

Campo numérico de tres posiciones y obligaorio, en el cual se registra el código de la garantia

Descipción

Campo alfanumérico en el cual se regista el nombre de la garantia

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificables es: Descripción.
