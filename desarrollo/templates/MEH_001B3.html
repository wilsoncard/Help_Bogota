Definición de bines

Cuentas de depósito por bin: Parámetros que aplicarán para cada uno de los tipos de cuenta asignados a cada Bin, que se utilizarán en el momento de capturar el tipo y número de cuenta, ya sea para una solicitud o una novedad de un producto con el Bin seleccionado. Es de aclarar que cuando en una opción se ingrese el tipo y número de cuenta, pero ésta no contenga un código de Bin, el sistema realizará la validación para cada tipo de cuenta, con base en los parámetros del Tipo de cuenta con Código 0 (Cero).

El formulario contiene las opciones Eliminar y Adicionar. Adicionalmente, se puede regresar a la opción Definición de Bines a través del botón volver de la parte superior izquierda.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Tipo de cuenta

Lista de valores que contiene tres opciones: Cuenta de ahorros, Cuenta corriente, y Rotativo.

Código cuenta

Lista de valores con campo alfanumérico de dos caracteres, en el cual se digita el código para asignar a la nueva cuenta, definida en la opción Tipos de cuenta.
