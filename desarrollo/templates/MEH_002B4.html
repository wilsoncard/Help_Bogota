Definición de productos - Otros plásticos

Esta opción que se invoca desde el formulario Definición de productos, le permite a la entidad adicionar o modificar la información correspondiente a plásticos adicionales que deben ser realzados y entregados al cliente conjuntamente con su tarjeta de crédito.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: si el usuario invoca la opción Adicionar se despliega un nuevo formulario con los siguientes campos.

Descripción de campos

Código

Campo obligatorio con lista de valores de la cual se puede seleccionar el plástico adicional que es necesario realzar y entregar a un cliente que se le otorgue una tarjeta perteneciente al producto que se está parametrizando.

Genera clave

Este campo tipo check permite indicar si para el plástico adicional a entregar al cliente se requiere generar clave para su utilización.

Actualizar: si el usuario selecciona un registro e invoca la opción Actualizar, se despliega un formulario en el cual el único campos modificable es Genera clave.
