Tipos documento identificación

Mediante esta función se habilita la consulta y mantenimiento de una tabla en la cual se almacenan los códigos de los diferentes tipos de documentos de identificación estandarizados por las entidades financieras; la forma permite asociarle a cada uno la clase de persona a la que corresponde, el código interno con que lo identifica el aplicativo, y si es el caso, el programa que los valida.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo numérico de dos dígitos, obligatorio, que corresponde al código asignado a cada uno de los tipos de documentos de identificación.

Nombre

En este campo alfanumérico de 30 posiciones, obligatorio, se registra el nombre asociado al código anterior.

Código interno

Campo que posee lista de valores que permite asociar cualesquiera tipo de documento definido en esta pantalla, con uno en particular relacionado con la programación interna del aplicativo, es decir, con el cual el programa lo reconocerá. Los posibles valores a elegir son: Cédula, Nit, Tarjeta identidad, Cédula extranjería, pasaporte, Seguro social, Fidecomiso, Sociedad extranjera sin nit, Nit extranjería, y Otro.

Abreviatura

En este campo alfanumérico de 5 posiciones, obligatorio, se registra el mnemónico o representación del nombre asociado al código de cada tipo de documento, utilizado para reportes impresos o consultas.

Naturaleza

Campo que contiene lista de valores de la cual se selecciona entre Natural o Jurídica la clase de persona asociada a cada tipo de documento de identificación.

Programa validador

En este campo se selecciona de la lista de valores adjunta, siempre y cuando aplique, el procedimiento que valida el dígito de chequeo del NIT o los rangos de tipo de documento de identificación según el sexo del cliente.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son: Nombre, Abreviatura, Naturaleza, y Programa validador.
