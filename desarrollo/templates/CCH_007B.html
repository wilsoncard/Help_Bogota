Actividades por tipo de perfil

Esta forma permite la creación o definición de los distintos perfiles a asociar a los funcionarios internos o de la casa de cobranza; así mismo, a cada perfil definido se le vinculan tanto los tipos de obligación que le es dable asumir, como los tipos de cobro que puede efectuar y las actividades de cobro que debe atender.

PerfilesEn este primer formulario la entidad puede crear o definir cuantos perfiles requiera ésta o la empresa externa de cobranzas, acorde con los tipos de cobro y tipos de obligación, puesto que puede ser necesaria una cualificación determinada, para atender adecuadamente las gestiones de cobro.

Filtro: El formulario cuenta con un filtro inicial de búsqueda, necesario para la consulta de la información:

Código

Campo alfanumérico de 15 posiciones que permite realizar un filtro de búsqueda por el código del perfil.

Descripción

A través de este campo alfanumérico de 50 posiciones es posible realizar la búsqueda por la descripción de los perfiles.

El formulario contiene los enlaces: Actualizar,Eliminar, Adicionar y Detalle. Adicionalmente cuenta con las opciones Actividades de cobro, Agentes y Metas.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo alfanumérico de 15 posiciones, obligatorio, permite asignar una referencia numérica o alfanumérica para identificar cada uno de los perfiles definidos por la entidad o casa de cobranza.

Descripción

En este campo alfanumérico de 50 posiciones, obligatorio, se indica el nombre o descriptivo asociado a cada uno de los códigos de perfil creados.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un formulario en cual los únicos campos modificables son los siguientes:

Detalle: Si el usuario invoca la opción Detalle se despliega el siguiente formulario.

Actividades de cobro: Grupo de datos que permiten asociarle a cada perfil por tipo de cobro las actividades que le es dable efectuar.

Tipo de cobro

Este campo cuenta con lista de valores poblada mediante la opción Tipos de cobro de este mismo módulo, de la que debe seleccionarse la clase de cobro que se asocia al tipo de obligación predefinida.

Actividad

Campo con lista de valores poblada mediante la opción Actividades de cobro de este mismo módulo, de la que debe seleccionarse todas y cada una de las actividades permitidas al perfil que se está definiendo y que se asocian al tipo de cobro y a la obligación predefinida.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un formulario en cual los únicos campos modificables son los siguientes:

Agentes: Este formulario muestra todos los agentes que se encuentren dentro del perfil selecccionado, de acuerdo a lo que se haya paremetrizado para los Agentes de cobro en la opción Operadores de cobro.

Detalle: Si el usuario invoca la opción Detalle se despliega el siguiente formulario.

Metas: A través de este formulario se pueden consultar las Metas que hayan sido parametrizadas para cada uno de los Perfiles.
