Programas comerciales

A través de esta opción la entidad puede determinar algunos beneficios adicionales para el tarjeta habiente, con el objetivo de convertirse en una herramienta de apoyo en los procesos de colocación masiva de productos, de acuerdo con el mercado objetivo y las características que la entidad define para lograr tal fin.

El formulario contiene los enlaces: Actualizar,Eliminar, Adicionar y Detalle. Adicionalmente, cuenta con un filtro de búsqueda y la opción Programas cambio.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código

Admite cualquier valor númerico de máximo 3 dígitos sin decimales.

Descripción

El campo Descripción, admite cualquier dato alfanúmerico de máximo 50 caracteres, y acepta mayúsculas y minúsculas.

Adicionar: Al activar ese enlace se despliega un formulario con los siguientes campos:

Código

En este campo obligatorio, se registra el código de los diferentes programas comerciales a definir.

Descripción

En este campo obligatorios se registra el nombre o descripción de los diferentes programas comerciales a definir.

Identificador

Permite marcar uno de los valores presentados, con los cuales se tendrán en cuenta los parámetros que aplicarán en el proceso de masificación de productos.

Edad: Permite definir si el programa comercial aplicará para aquellos clientes que cumplan con las características de la edad que se hayan definido.

Antigüedad: Los parámetros también se pueden definir por antigüedad de la tarjeta, tomada a partir de la fecha de vigencia o activación. Cuando se selecciona este campo, automáticamente se deshabilita la opción Programas cambio.

Edad inicial

Campo numérico de 3 posiciones, obligatorio, en el que se registra el rango inicial de edad del cliente a partir del cual se realiza el traslado del producto por efectos de aplicación del programa comercial, siempre y cuando se haya seleccionado este parámetro como identificador.

Edad final

Campo numérico de 3 posiciones, obligatorio, en el que se registra el rango final de edad del cliente hasta el cual se realiza el traslado del producto por efectos de aplicación del programa comercial, siempre y cuando se haya seleccionado este parámetro como identificador.

% Manejo / Afiliación / Seguro

Campos numéricos de 3 posiciones enteras y 2 decimales, obligatorios, en los cuales se define la proporción que se cobrará al cliente por cualquiera de los tres conceptos, sobre los valores definidos para el producto, por pertenecer a un determinado programa comercial. Estos parámetros son aplicados jerárquicamente a partir de lo definido en prioridad cobro de cuotas tanto para productos débito como crédito.

Exención

Campo numérico no obligatorio en el que se puede indicar el número de periodos durante los cuales aplican las condiciones de %manejo / seguro / afiliación definidas en el campo anterior.

Nro. mora 1/ Nro. mora 2 /Nro. mora 3

En este campo obligatorio, la entidad establece el número de moras máximo para determinada altura, que puede presentar un cliente para que sea susceptible de efectuársele un cambio de producto.

Períodos Evaluar

Este campo obligatorio, permite registrar la cantidad de facturaciones desde la última hacia atrás, que servirán como base para evaluar el número de veces en las que el cliente ha entrado en mora, y poder determinar así que su producto de un programa comercial pase a uno de manejo de tarjeta principal común y corriente.

Edad Traslado

Corresponde a la edad que debe alcanzar el cliente para que el producto de un programa comercial sea cambiado a uno de tarjeta principal común y corriente. Es un campo numérico de 3 posiciones y su diligenciamiento no es requerido.

% Aumento para línea principal

En este campo se define la proporción de aumento de cupo sobre la línea principal del nuevo producto, teniendo en cuenta que no se supere el cupo máximo para dicha línea.

Actualizar: Al activar ese enlace , se despliega un formulario en el cual todos los campos son campos modificables si el identificador es por antigüedad y si es por edad, el único campo no modificable es Código.
