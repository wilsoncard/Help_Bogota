Premios puntajes

Mediante esta función se pueden definir los incentivos o premios a los cuales el cliente se hace acreedor, al acumular puntos por los consumos efectuados por este, en los establecimientos que pertenecen a cualquiera de los distintos esquemas de puntaje definidos por la entidad. Dichos premios pueden estar representados en: electrodomésticos, pasajes aéreos, beneficios o abonos a sus tarjetas, entre otros.

Filtro: El formulario cuenta con un filtro inicial de búsqueda:

Código

Campo numérico de máximo 7 dígitos, mediante el cual, digitando el código del premio, se puede realizar la consulta.
Descripción

Campo alfanumérico de hasta 60 posiciones, mediante el cual, digitando el nombre del premio o parte de él, se puede realizar la consulta.

El formulario contiene las opciones Actualizar, Eliminar, Adicionar y Detalle.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo numérico de máximo 7 dígitos, obligatorio, en el que se registra el código asignado a cada uno de los premios que puede obtener el cliente, y el que servirá de referencia para que la entidad pueda ejercer control sobre las existencias de los mismos.

Descripción

Una vez registrado el código del premio, en este campo alfanumérico de hasta 60 posiciones, obligatorio, se registra el nombre o descriptivo asociado a cada código.

Saldo inventario

Campo en cual se registra la cantidad de unidades disponibles por clase de premio. Este campo luego de ser parametrizado, se actualiza tanto por la redención de premios hechas por los clientes como cuando éstos son reversados.

Cantidad de reorden

En este campo numérico de 10 dígitos, no obligatorio, se define el saldo mínimo de unidades que la entidad desea tener disponibles para ser entregadas a los clientes, con el fin de asegurar las existencias suficientes para cumplir la demanda de los mismos. De acuerdo con la cantidad expresada en este campo la entidad podrá realizar un nuevo pedido de éstos si así lo requiere para lo cual el sistema generará un mensaje de advertencia en la opción redención premios cuando el saldo de unidades ha alcanzado el valor registrado en este campo o incluso si es inferior.

Puntos requeridos

Campo numérico de 7 dígitos, obligatorio, en el que se registra la cantidad de puntos que necesita un cliente para poder hacerse acreedor a cada premio parametrizado.

Plan millas

En este campo se define si el premio previamente registrado corresponde a millas o no, lo que determina que éstas se puedean revertir a través de la opción Reversión premios.

Referencia millas

En este campo la entidad inscribe las características propias del premio que se ha registrado previamente tales como código de referencia, serie o cualquier dato que identifique dicho producto.
