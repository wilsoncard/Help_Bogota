Categorías de productos

A través de este formulario, la entidad puede parametrizar la clasificación o estratificación que se van a asociar con lo diferentes productos ofrecidos y que le permiten generar informes consolidados con destino a las áreas de planeación y comercial.

Por medio del filtro provisto, el usuario tiene la posibilidad de consultar información específica bien sea por el códio de la categoría, la descripción o el tipo de producto CPS.

El formulario contiene los enlaces: Actualizar, Eliminar y Adicionar.

Adicionar: si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos.

Descripción de campos

Código

Campo alfanumérico de 10 caracteres, obligatorio, en el cual se registra el código que identifica cada una de las diferentes categorías de productos definidas por la entidad.

Nombre

Este campo alfanumérico de 30 posiciones, obligatorio, permite registrar el nombre de la categoría asociada con el código anterior.

Señal avalado

Campo obligatorio tipo combo en el cual se puede seleccionar entre Debe buscar amparado o No busca amparado para indicarle al sistema si en el caso de tarjetas principales con dependientes debe o no buscar las mismas para enviar la información en las interfases respectivas.

Producto CPS

En este campo obligatorio tipo combo, se puede seleccionar entre Tarjeta de crédito y Otros créditos para indicar a qué tipo de producto corresponde la categoría parametrizada.

Actualizar: si el usuario selecciona un registro e invoca la opción Actualizar, se despliega un formulario en el cual el único campo NO modificable es el Código.
