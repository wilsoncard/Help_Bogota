Franquicias

Parámetros regiones para el exception file: Conjunto de campos que permiten señalar para cuales de las regiones definidas por cada franquicia, aplican las novedades reportadas dentro del archivo de exception file por tarjeta o por cliente.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar. Adicionalmente, cuenta con un botón en la parte superior para regresar al formulario principal de Franquicias.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes bloques:

Código

Campo obligatorio en el que se registra cada unos de los códigos, definidos por cada franquicia, que identifica las diferentes zonas para las que es posible restringir el uso de las tarjetas, cuyas novedades viajan en el archivo de exception file.

Descripción

Campo dentro del que se relaciona el nombre del país, estado o región asociado al código.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es la Descripción.
