Definición de bines

Grupos afinidad: Tercer y último nivel de la parametrización, en el que se relacionan los grupos que presentan características comunes, lo que posibilita el otorgarles algunos beneficios a los clientes que pertenezcan a estos, con el objetivo de incentivar el consumo; para los grupos que aplique, se le puede asociar un programa definido por Visa, al generar las estadísticas de operaciones para la misma, el sistema pueda clasificar correctamente la información a generar. Además de: Actualizar, Eliminar y Adicionar, el formulario cuenta con otros dos enlaces: Definición de Bines y Segmentos.

Adicionar: Al activar ese enlace se despliega un formulario:

Grupos afinidad

Dupla de campos obligatorios, el primero, alfanumérico de tres dígitos, debe contener el código asignado al nuevo grupo. El segundo, alfanumérico de 30 posiciones, debe contener el nombre asociado al código del nuevo grupo.

Programa

Campo no obligatorio, con lista de valores del que se puede seleccionar, si aplica, el programa de afinidad establecido por Visa, para asociarlo a cada uno de los grupos creados.
