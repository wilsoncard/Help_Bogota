Consulta Bloqueos

A través de este formulario y usando los datos básicos de identificación de sus clientes, la entidad puede efectuar una consulta a su base de datos, y conocer el estatus en cuanto a bloqueos se refiere, de cada una de las tarjetas asociadas a los datos de identificación utilizados. En el bloque Resultados, el sistema muestra los diferentes registros de tarjetas sin tener en consideración que tengan o no bloqueos vigentes.

Tipo identificación

Campo que posee lista de valores tipo combo, con las distintas opciones de documento de identificación, de la que debe seleccionar el tipo que corresponda al cliente para el que se va a consultar el estatus de los bloqueos de la(s) tarjetas(s) a su nombre.

Número identificación

Campo en el que debe ingresar, con o sin ceros a la izquierda, el número del documento de identificación del cliente a consultar.

Resultados

Número Tarjeta

Campo en el que se muestra el número, enmascarado, de cada una de las tarjetas a nombre del cliente consultado.

Código interno

Campo que muestra el código interno de cada una de las tarjetas a nombre del cliente consultado.

Nombre Plástico

Campo en el que se ilustra el nombre del cliente tal y como aparece en el plástico de cada una de las tarjetas a nombre del cliente consultado.

Bloqueo

Campo en el que se muestra el bloqueo detentado en cada una de las tarjetas a nombre del cliente consultado, dato que puede ser nulo.

Fecha Bloqueo

Campo que en formato DD-MM-AAAA HH:MM:SS ilustra, la fecha del sistema en la que se aplicó el bloqueo correspondiente.

Tipo Tarjeta

Campo que muestra el tipo de tarjeta a la que corresponde cada uno de los registros recuperados.
