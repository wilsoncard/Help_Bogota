Requisitos

Esta función habilita la consulta y/o mantenimiento de la tabla que almacena los requisitos que la entidad requiere sean solicitados a los clientes de la entidad cuando efectúen un reclamo.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar. Adicionalmente, cuenta con un filtro de búsqueda.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código

Campo numérico de cuatro dígitos, que contiene el código del requisito que se desea buscar.

Descripción

En este campo alfanumérico de máximo 100 caracteres, se ingresa la definición asociada al requisito buscado.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo numérico de cuatro dígitos, obligatorio, que contiene el código asignado a cada requisito, el cual lo identifica dentro del sistema.

Descripción

Campo alfanumérico de 100 posiciones, obligatorio, en el cual se registra el nombre de cada requisito asociado al respectivo código.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es la Descripción.
