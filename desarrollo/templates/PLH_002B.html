Generación de pedidos

Mediante esta opción que se invoca desde el formulario Inventario de plásticos se generan los pedidos de plásticos a los proveedores, de acuerdo con la cantidad de reorden establecida y con las necesidades propias de la entidad.

Al ingresar a la opción, se despliega un wizard de dos pasos que permite la generación de pedidos para los diferentes modelos de plásticos que hayan sido parametrizados por la entidad.

Descripción de campos

Modelo

Campo obligatorio con lista de valores que permite seleccionar el modelo de plástico para el que se desea generar un pedido al proveedor.

Proveedor

Este campo obligatorio despliega el nombre del proveedor del plástico y a quien se le va a generar el pedido.

Cantidad

Campo obligatorio en el que se registra la cantidad de plásticos a solicitarle al proveedor. Por defecto después de haber seleccionado el modelo, este campo se llena automáticamente con la cantidad de reorden parametrizada para el mismo pero puede ser modificada por el usuario.

Después de haber ingresado el pedido para un modelo de plástico, mediante el segundo botón del wizard se van adicionando uno a uno los pedidos y luego al desplazarse a la segunda página se muestra el resumen de todos desplegando para cada uno el modelo, el proveedor y la cantidad pedida. El formulario cuenta con un check que permite marcar aquellos pedidos que ya no se desea generar.

Una vez grabados los registros el sistema le genera internamente a cada pedido una secuencia y actualiza automáticamente para cada modelo el campo 'Pedido' del formulario Inventario de plásticos .

El sistema controla que para un mismo modelo no se pueda hacer más de un pedido a la vez y de igual manera que si el modelo ya tiene un pedido anterior este tenga registrada por lo menos alguna entrega parcial.
