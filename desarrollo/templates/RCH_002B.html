Roles

Mediante esta función se habilita la consulta y mantenimiento de la tabla en la cual se almacenan los roles de los gestores que podrán atender los diferentes tipos de reclamos existentes.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar. Adicionalmente, cuenta con un filtro de búsqueda y la opción Gestores.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código

Campo alfanumérico de 16 caracteres que contiene el código del rol que se desea buscar.

Descripción

En este campo alfanumérico de máximo 30 caracteres, se ingresa la definición asociada a cada código de rol buscado.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código rol

Campo alfanumérico de 16 caracteres, obligatorio, que contiene el código asignado a cada rol y que lo identifica dentro del sistema.

Descripción

Campo alfanumérico de 30 posiciones, obligatorio, en el cual se registra el nombre de cada rol asociado al respectivo código.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es la Descripción.

Gestores: A través de este hipervínculo, se pueden consultar los gestores relacionados con cada uno de los roles creados según lo parametrizado en la opción Gestores.
