Monedas

Mediante esta opción se posibilita la consulta y/o mantenimiento de la tabla que contiene la información básica de las monedas oficialmente utilizadas por las franquicias, principalmente para el manejo de las transacciones realizadas en el extranjero. El formulario cuenta con una opción de Filtro de Búsqueda por Código y Nombre.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Actualizar se despliega el siguiente formulario:

Código

Campo numérico de tres dígitos, obligatorio, asignado a cada una de las monedas oficialmente utilizadas dentro de las franquicias.

Nombre

Campo alfanumérico de 30 posiciones, obligatorio, en el cual se registra el nombre asociado al código de cada moneda.

Abreviatura

Campo alfanumérico de 6 posiciones, obligatorio, en el cual se registra el nombre resumido o mnemónico para utilizarse en los reportes.

Símbolo

Campo alfanumérico de 4 posiciones, no obligatorio, contiene la representación de cada moneda para ser utilizada en los reportes.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son: Nombre, Abreviatura, Símbolo.
