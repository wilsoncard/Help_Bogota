Gestores

Mediante esta función se habilita la consulta y mantenimiento de la tabla en la cual se almacenan los diferentes gestores que podrán atender los diferentes tipos de reclamos existentes. Así mismo, éstos se relacionarán con los roles ya creados y se podrán visualizar las estadísticas de la asignación y atención de dichos reclamos por cada gestor existente.

El formulario contiene las opciones Actualizar, Eliminar, Adicionar y Detalle. Adicionalmente, cuenta con un filtro de búsqueda y las opciones Roles y Estadísticas.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código

Campo numérico de 4 dígitos que contiene el código del gestor que se desea buscar.
Nombre

Campo alfanumérico de 60 posiciones, que contiene el nombre del gestor que se desea buscar.
Tipo de identificación

En este campo tipo combo, el usuario puede seleccionar el tipo de documento asociado al gestor que se desea consultar en la base de datos.
Número de identificación

Campo numérico de 16 posiciones, en el que se ingresa el número de identificación del gestor que se quiere consultar.
Tipo de relación

En este campo tipo combo, el usuario puede seleccionar entre Interno o Externo, según la característica del gestor que se desea consultar en la base de datos.
Activo

En este campo tipo combo, el usuario puede seleccionar entre Si o No, según si el gestor que se desea consultar en la base de datos se encuentra activo o no.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código gestor

Campo numérico de 4 dígitos, obligatorio, que contiene el código asignado a cada gestor y que lo identifica dentro del sistema.

Tipo de identificación

En este campo campo tipo combo obligatorio, el usuario puede seleccionar el tipo de documento asociado a cada gestor.
Número de identificación

Campo numérico de 16 dígitos obligatorio, en el que se ingresa el número de identificación del gestor asociado al tipo de documento seleccionado en el campo anterior.
Nombre

Campo alfanumérico de 60 posiciones, obligatorio, en el cual se registra el nombre del gestor.
Tipo de relación

Campo tipo combo obligatorio, en el cual el usuario puede seleccionar entre interno o externo según si el gestor pertenece directamente a la entidad o trabaja de manera externa con ésta.
Usuario del sistema

En este campo obligatorio, se selecciona de la lista de valores el usuario al cual se asocia el gestor.
Dirección

Campo alfanumérico de 60 posiciones, obligatorio, en el cual se registra la dirección correspondiente al gestor.
Teléfono del contacto

Campo alfanumérico de 30 posiciones, obligatorio, en el cual se registra el teléfono correspondiente al gestor.
Teléfono celular

Campo alfanumérico de 30 posiciones, en el cual se registra el teléfono celular correspondiente al gestor.
E-mail

Campo alfanumérico de 30 posiciones, en el cual se registra la dirección electrónica o e-mail del gestor.
Activo

Campo tipo combo obligatorio, en el cual el usuario selecciona si el gestor se encuentra activo o no para tramitar reclamos.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual todos los campos son modificables excepto el código del gestor.
