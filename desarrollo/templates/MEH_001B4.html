Líneas de consumo

A través de esta opción se relacionan las distintas líneas de crédito ofrecidas por la entidad y que pueden ser asignadas o no a cada producto.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Línea de crédito

Campo obligatorio alfanumérico de dos dígitos, que permite asignar una identificación a cada línea de crédito con las cuales operará el producto.

Nombre

En este campo obligatorio se registra el nombre de cada línea de crédito una vez asignados sus correspondientes códigos.

Concepto

Campo obligatorio que permite asignar para cada línea de crédito definida por la entidad, el uso de la misma seleccionado de la lista de valores adjunta y que puede ser Compras, Avances en efectivo u Otros; adicionalmente, éste último puede ser asociado a líneas especiales tales como Salud, Turismo, Educación, etc.

Posición

En este campo numérico de 3 dígitos, obligatorio, se define para cada una de las líneas de crédito, el orden de prioridad o importancia para establecer jerarquías entre ellas de acuerdo con cada uno de los productos definidos.

Número de cuotas revolventes

Campo numérico de 3 posiciones, no obligatorio, en el cual se registra la cantidad de cuotas revolventes en las que por defecto se difiere el saldo del cliente si el sistema de cálculo del pago mínimo es revolvente.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son: Nombre, Concepto y Posición.
