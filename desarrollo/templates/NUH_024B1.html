Mensajes

Mensajes por lenguaje: A través de este formulario se ingresan todos los mensajes que se consideren necesarios para mostrar en los diferentes errores y advertencias que muestra el sistema. El ingreso se hace por cada uno de los lenguajes creados.

El formulario muestra las opciones Actualizar, Eliminar y Adicionar. Adicionalmente cuenta con un botón para regresar al formulario principal.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código mensaje

Campo que debe aceptar hasta ocho caracteres alfanuméricos.

Descripción

Campo que debe aceptar hasta 40 caracteres alfanuméricos. Permite realizar la búsqueda con el símbolo %.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código mensaje

Campo numérico de 5 posiciones, obligatorio, en el que se ingresa el código con el cual se identificará el error en el sistema.

Tipo

Lista de valores que permite determinar qué tipo de mensaje se está adicionando. Muestra las opciones OpenCard, Oracle y Java.

Descripción

El campo permite ser diligenciado con cualquier dato alfanumérico de máximo 500 caracteres, permitiendo mayúsculas y minúsculas.

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual sólo podrá actualizarse la Descripción.
