Información de establecimientos - Secciones

Desde este formulario, que se invoca mediante el enlace Secciones contenido en el formulario de Información de establecimientos, se registran la información detallada, acerca de la forma como está dividido o estructurado el establecimiento desde el punto de vista comercial y los POS que tiene habilitados, información que es utilizada al momento de definir las Estrategias de mercadeo.

El formulario contiene el hipervínculo Puntos de venta en el cual se parametriza la información de los POS del establecimiento.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: si el usuario invoca la opción Adicionar se despliega un nuevo formulario con los siguientes campos:

Descripción de campos

Código

Campo alfanumérico de máximo 6 posiciones, obligatorio, en el cual se registra el código asignado para identificar cada una de las diferentes secciones que componen el establecimiento.

Descripción

En este campo alfanumérico de máximo 60 posiciones, obligatorio, se registra el nombre o breve descripción de la sección asociada al código anterior.

Actualizar: si el usuario selecciona un registro e invoca la opción Actualizar se despliega un formulario en el cual el único campo modificable es la Descripción.

Puntos de venta : cuando el usuario selecciona un registro e invoca el hipervínculo Puntos de venta, se despliega un formulario en el cual se muestra la información de los POS pertenecientes al establecimiento.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: si el usuario invoca la opción Adicionar se despliega un nuevo formulario con los siguientes campos:

Descripción de campos

Número de terminal

Campo alfanumérico de hasta 16 posiciones en el que se registra el código o serie que identifica cada uno de los POS habilitados por el establecimiento.

Descripción

En este campo alfanumérico de máximo 60 caracteres, se registra el nombre o descripción del POS asociado al campo anterior.

Código de tienda

Campo numérico de hasta 11 dígitos en el que se registra el código que identifica la tienda o sucursal del establecimiento a la que pertenece el POS.

Bin establecimiento

En este campo alfanumérico de máximo 11 posiciones se registra si es del caso, el bin que identifica al establecimiento, cuando por ejemplo se trate de tarjetas de marca privada (retail).

Rango terminal

Campo tipo combo en el que se puede seleccionar entre: Nulo, Marca, Bajo o Alto.

Rango terminal

Campo tipo combo en el que se puede seleccionar entre: Nulo, Marca, Bajo o Alto.

Ultima actualización

En este campo obligatorio, en el que, mediante la funcionalidad de un calendario, se registra la fecha en la cual se hizo la más reciente actualización de datos para la cuenta.
