Actividades de cobro

Si la entidad y/o la casa de cobranza requieren de una herramienta que sirva como mecanismo de control de gestión y/o para efectuar el pago a los funcionarios por las labores realizadas, debe parametrizar por medio de esta forma todo el proceso de cobranza de principio a fin, tan desagregado como se desee.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

Código

Campo numérico de 3 dígitos, obligatorio, que identifica a cada una de las etapas o pasos que componen el proceso de una cobranza.

Descripción

En este campo alfanumérico de 50 posiciones, obligatorio, se registra el nombre o descriptivo asociado a cada uno de los códigos de las actividades definidas.
